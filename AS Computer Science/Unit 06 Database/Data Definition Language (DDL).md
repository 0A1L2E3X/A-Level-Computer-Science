
- creating database
~~~sql
CREATE DATABASE <'database name'>
~~~

- creating table
~~~sql
CREATE TABLE <'table name'>
~~~

- alter table
~~~sql
ALTER TABLE <'table name'>
~~~

- add keys (primary & foreign)
~~~sql
PRIMARY KEY <'field name'>

FOREIGN KEY <'field name'> REFREENCES <'table name'>('field name')
~~~
